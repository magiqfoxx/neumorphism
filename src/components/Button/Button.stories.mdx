# Button

With `MDX` we can define a story for `Button` right in the middle of our
markdown documentation.

<Preview>
  <Story name="text only">
      <Button variant={select("variant", listOfVariants, 'light')}>{text("text", "Button")}</Button>
  </Story>
</Preview>
